Overview

The goal of this project is to create a working front end of a shopping cart using HTML, CSS and JavaScript. We first had to list the products, then create the functionality to add and remove them to/from the cart. We also needed to calculate the total price based on the items in the cart as well as implement a simple discount code system.

Extra Details

This project really opened a lot of doors that were previously closed in the world of JS. A concept that stood on to me particularly was the 'source of truth'. Here we define one location that was a source of truth for all interactions with the application. My source of truth was an array of products from which I was able to add them to the cart and calculate totals based on their prices.
